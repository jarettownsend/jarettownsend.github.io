This project is mostly a data engineering project with a little bit of statistical analysis. The general idea behind it is if one team is significantly better than another, it's naive to assume they'll lose just because they're playing in unfamiliar weather. Instead of making raw predictions, this model builds on the betting lines set by sportsbooksâ€”who already factor in many variables and tend to predict game margins fairly well. The assumption here is that sportsbooks may undervalue weather differences. So, if a team is expected to win by 10 points but is traveling to a location with unfamiliar weather, this model would adjust that margin slightly downward. In short, it adds a small weather-based correction on top of the sportsbook's original prediction using a few hard-coded rules.
